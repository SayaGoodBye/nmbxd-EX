## X岛-EX：网页端增强，移动端般的浏览体验

如你所见，这是一个适配网页端[**X岛揭示板**](https://www.nmbxd1.com/)的[油猴脚本](https://greasyfork.org/zh-CN/scripts/531005-x%E5%B2%9B-ex)，开发目的即为“增强网页端，赋予其移动端般的浏览体验”。

### 主要功能：
- 快捷切换饼干
- 添加页首页码
- 关闭图片水印
- 预览真实饼干
- 隐藏无标题/无名氏/版规
- 显示外部图床
- 自动刷新饼干
- 无缝翻页
- 默认原图+控件
- 新标签打开串
- 优化引用弹窗
- 拓展引用格式
- 当页回复编号
- 扩展坞增强
- 回复浮窗配套快捷键
- 拦截回复中间页
- 颜文字拓展
- 标记Po主
- 发串UI调整
- 标记/屏蔽/只看-饼干/关键词：折叠/隐藏/批注模式
- 增强X岛匿名版
- 板块页快速回复
- 允许展开板块页长串
- 野生搜索酱
- 收起侧边栏
- 图片隐藏：默认/模糊/无图/Tips模式
- 图片自动压缩/非法图像格式（无GCT）GIF重编码
- 链接自动识别
- 设置项导入/导出-剪贴板/文件
- 图片右键菜单-可复制GIF/APNG
- 侧边栏常用串
- 侧边栏/设置面板浏览历史
- 我的发言（发串/回复）

鼠标悬停至设置面板的每个选项上可以查看功能简介，或者可在[串内](https://www.nmbxd1.com/t/67024789)查看，此处不过多赘述。



### 浏览器拓展版安装方法：

1. 打开浏览器拓展页面
2. 开启“开发者模式”
3. 点击“加载解压缩的拓展”
4. 选择通过[GitHub release](https://github.com/SayaGoodBye/nmbxd-EX/releases/latest)或者[百度网盘](https://pan.baidu.com/s/1-ELWglsTXG8jK5S6WwqtsQ?pwd=k8zf)下载的拓展压缩包（nmbxd-EX-Extension-version.zip）解压后的文件夹
5. 安装成功\\( ﾟ∀。)/

PS:本浏览器拓展适配Chrome、Edge等Chromium内核浏览器，**Firefox浏览器无法使用**
